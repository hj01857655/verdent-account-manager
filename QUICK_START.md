# ⚡ 快速启动指南

## 🎯 5分钟上手 Verdent

### 第1步：下载安装（1分钟）

#### Windows 用户
1. 下载 `Verdent账号管理器_1.2.0_x64.msi`
2. 双击安装
3. 桌面找到图标启动

#### Mac 用户
1. 下载 `Verdent账号管理器_1.2.0.dmg`
2. 拖动到 Applications
3. 启动器中打开

---

### 第2步：获取第一个账号（2分钟）

#### 方法A：自动注册（推荐）
1. 点击右上角 **"自动注册"** 按钮
2. 保持默认设置（1个账号）
3. 点击 **"开始注册"**
4. 等待约30秒完成

#### 方法B：导入已有账号
1. 点击 **"导入账号"** 按钮
2. 粘贴你的 Token 或账号密码
3. 点击 **"导入"**

---

### 第3步：登录到编辑器（1分钟）

1. 找到刚添加的账号卡片
2. 点击对应的编辑器按钮：
   - 🆚 **VS Code**
   - 🔷 **Cursor**
   - 🌊 **Windsurf**
3. 编辑器会自动打开并完成登录

---

## 🔥 常用操作速查

| 操作 | 方法 | 快捷键 |
|------|------|--------|
| 全选账号 | 点击"全选"按钮 | Ctrl+A |
| 批量刷新 | 选中账号 → 点击"刷新" | - |
| 导出数据 | 点击"导出" → 选择格式 | - |
| 隐私模式 | 点击"隐私模式"按钮 | - |
| 筛选账号 | 点击统计卡片 | - |

## 💡 实用技巧

### 批量注册账号
```
自动注册 → 设置数量为 5 → 并发数设为 2 → 开始注册
```

### 快速筛选可用账号
```
点击 "满额度账号" 卡片 → 只显示未使用的账号
```

### 批量导出 Token
```
全选 → 导出 → 选择 TXT 格式 → 获得 Token 列表
```

### 保护隐私
```
开启隐私模式 → 邮箱自动打码显示
```

## ⚠️ 注意事项

1. **首次使用** Windows 可能需要安装 WebView2
2. **自动注册** 使用免费临时邮箱，24小时后失效
3. **数据备份** 定期导出账号数据以防丢失
4. **隐私安全** 不要分享包含 Token 的截图

## 🆘 遇到问题？

### 无法启动？
- Windows: 安装 [WebView2](https://developer.microsoft.com/microsoft-edge/webview2/)
- Mac: 系统偏好设置 → 安全性与隐私 → 允许运行

### 自动注册失败？
- 检查网络连接
- 减少并发数为 1
- 确保 Chrome 已安装
