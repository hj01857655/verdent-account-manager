## [1.0.0] - 2024-11-16

### 🎉 首次发布

#### ✨ 新功能
- **账号管理**
  - 集中管理所有 Verdent AI 账号
  - 支持增删改查基本操作
  - 账号信息实时刷新
  
- **自动注册**
  - 批量自动注册新账号（1-10个）
  - 使用免费临时邮箱服务
  - 支持并发注册提高效率
  - 可选随机密码或固定密码
  
- **批量操作**
  - 全选/多选账号
  - 批量刷新账号信息
  - 批量导出账号数据
  - 批量删除账号
  
- **一键登录**
  - VS Code 快速登录
  - Cursor 快速登录
  - Windsurf 快速登录
  - 使用 PKCE 安全认证
  
- **导入导出**
  - 支持 Token 导入
  - 支持账号密码导入
  - 支持 JSON 批量导入
  - 导出格式：JSON、CSV、TXT
  
- **智能筛选**
  - 按额度状态筛选
  - 满额度账号筛选
  - 已消耗账号筛选
  - 异常账号筛选
  
- **隐私保护**
  - 邮箱隐私模式
  - 敏感信息自动打码
  - 编辑时显示原始信息
  
- **数据统计**
  - 实时统计面板
  - 总账号数统计
  - 额度使用统计
  - 账号状态分布

#### 🎨 界面设计
- 采用苹果设计风格
- 响应式布局设计
- 流畅的动画效果
- 现代化 UI 组件

#### 🔧 技术特性
- 基于 Tauri + Vue 3
- Rust 后端高性能
- 本地数据持久化
- 跨平台支持（Windows/macOS/Linux）

#### 📦 构建部署
- GitHub Actions 自动构建
- 支持所有主流平台
- 一键打包脚本
- 自动发布 Release
